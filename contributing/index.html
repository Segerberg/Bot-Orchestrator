Contributing

Did you find a bug?

  • Make sure that the bug hasn't already been reported.

  • If no issue describing the same problem exists, open a new one with as much information as possible:

    • A title along with a description that explains the problem as clear as possible
    • A code sample or test case demonstrating the unexpected behavior

Do you have a new feature or a fix for a bug?

  • Open a new GitHub pull request with the update

  • Reference any applicable issues
